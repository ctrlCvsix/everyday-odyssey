# Everyday Odyssey Prototype

Unity project path: `D:\UnityProjects\EverydayOdyssey`

Prototype controls:

- `WASD`: move
- `Space`: jump
- `V`: switch first-person / third-person
- `Mouse0`: send laptop code attack
- `E`: interact with review terminals and upload gate

Current prototype content:

- Chinese / Korean language switching
- opening tutorial panel with rules before gameplay starts
- 5 teacher enemies with imported Kenney character models
- student character with imported model and laptop code attack
- random code strings on projectiles (`py`, `java`, `c`-style snippets)
- hit feedback, flash reaction, upgraded UI, and sound hooks
- Shinhan University inspired entrance with twin black pillars and imported building / road assets
- collision boundaries to prevent wall clipping
- creative win condition: hack 5 mentor review terminals, then upload the final defense build between the twin pillars before the deadline

To regenerate the scene in Unity:

1. Open the project.
2. Run `Everyday Odyssey > Build Prototype Scene`.
